## Overview

Anvsoft SynciOS Data Recovery is a Windows recovery tool for retrieving information from iPhone, iPad, and iPod devices. It is intended for situations where files, messages, or application data must be located through a supported iOS device or backup source.

Recovery can be performed from a connected device, an iTunes backup, or an iCloud backup. The application lets you inspect discovered content before recovery and select only the records or files that should be restored.

Yes. The preview feature allows you to inspect the detected content before deciding what to export.

**Can I restore only specific items?**  
Yes. Individual files or records can be selected, allowing you to avoid recovering the complete result set.

**How are the application settings managed?**  
Settings are changed within the application. Local preference files, when present, are generally stored beside the application or in the Windows user profile directory.

**What can I try if my device is not recognized?**
